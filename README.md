# üåç Global Regulatory Bodies Dataset

This repository contains a structured, comprehensive dataset cataloging key regulatory and oversight bodies across the **United Kingdom (UK)**, **Republic of Ireland (IE)**, and the **United States of America (USA)**.

The data is organized by country and sector, providing essential information for understanding the compliance landscape for consumers and businesses in Finance, Health, Data Privacy, and Transport.

## üìÅ Dataset Structure

The core data is presented in a single table (or CSV/JSON format, typically named `regulatory_bodies.csv`), featuring the following columns:

| Column Name | Description | Data Type |
| :--- | :--- | :--- |
| `Country` | The jurisdiction the body operates in (IE, UK, USA). | String |
| `Sector` | The industry or area of focus (e.g., Finance/Banking, Data Protection, Aviation). | String |
| `Regulator_Council` | The official name of the regulatory or oversight body. | String |
| `Primary_Focus` | Identifies the main constituency the body serves: **Consumer**, **Business**, or **Both**. | String |
| `Website` | The official URL for the regulatory body. | String |

## üí° Key Sectors Covered

The dataset currently covers the following high-impact regulatory sectors:

1.  **Finance/Banking:** Oversight of financial stability, market conduct, and consumer lending/savings.
2.  **Medical/Health:** Regulation of health products, medical devices, and professional healthcare standards.
3.  **Data Protection:** Enforcement of data privacy and information rights (e.g., GDPR, CCPA).
4.  **Aviation/Transport:** Regulation of air safety, transport infrastructure, and operator licensing.

## üéØ Use Cases

* **Compliance Mapping:** Quickly identify relevant regulatory authorities when expanding business operations into a new jurisdiction.
* **Legal Research:** Serve as a foundational list for in-depth legal and comparative analysis across different common law jurisdictions.
* **Consumer Advocacy:** Identify the correct organization for consumers to submit complaints or seek information on a specific industry.
* **Educational Resource:** Use as a teaching tool to illustrate comparative regulatory governance models.
